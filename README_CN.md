# COMP9001 期末项目 - 元素魔法大乱斗

[English Version](./README.md) | 中文版

## 项目简介

这是悉尼大学（University of Sydney）COMP9001课程的期末项目。本项目是一个**元素魔法大乱斗**游戏 - 一款基于粒子系统的魔法防御游戏，玩家使用元素魔法击退不断涌来的敌人。

## 课程信息

- **学校**: 悉尼大学 (University of Sydney)
- **课程代码**: COMP9001
- **项目类型**: Final Project (期末项目)
- **项目名称**: Elemental Magic Arena (元素魔法大乱斗)
- **提交截止**: 2025年11月2日

## 项目特色

### 🎮 游戏概念
玩家扮演一位魔法师，用鼠标绘制元素魔法来击败从上方进攻的敌人。不同元素之间有克制关系，组合元素可以释放强大的连招技能！

### 核心功能
- 🔥 **5种元素魔法**: 火、水、雷、冰、风
- 💫 **粒子特效**: 震撼的粒子系统，包含拖尾和发光效果
- 👹 **敌人系统**: 5种敌人类型，各有元素属性和弱点
- ⚔️ **连招技能**: 组合元素创造8+种强力魔法连招
- 📈 **成长系统**: 升级、解锁技能、提升属性
- 🎯 **波次系统**: 难度递增，包含Boss战
- ✨ **视觉效果**: 爆炸、屏幕震动、慢动作等

### 技术亮点
- 🖥️ **PyQt5构建**: 现代GUI，自定义粒子渲染
- 🎨 **高级粒子系统**: 数千粒子的物理模拟
- ⚡ **实时战斗**: 60 FPS流畅游戏体验
- 🎮 **游戏机制**: 元素交互、碰撞检测、敌人AI
- 🌟 **视觉打磨**: Bloom效果、运动模糊、屏幕特效


2. **安装Python依赖**
   ```bash
   pip install -r requirements.txt
   ```

```bash
python main.py
```

## 游戏操作

### 鼠标控制
- **鼠标左键+拖拽**: 绘制元素魔法
- **鼠标移动**: 瞄准魔法

### 键盘控制
- **1 / Q**: 切换到火元素 🔥
- **2 / W**: 切换到水元素 💧
- **3 / E**: 切换到雷元素 ⚡
- **4 / R**: 切换到冰元素 ❄️
- **5 / T**: 切换到风元素 🌪️
- **Space**: 使用连招技能（准备就绪时）
- **Shift**: 防御姿态
- **Tab**: 查看技能树
- **Esc**: 暂停菜单
- **[ / ]**: 调整时间速度（调试用）

## 游戏玩法

### 游戏目标
通过使用元素魔法击败敌人，尽可能生存更多波次。使用正确的元素来克制敌人的弱点，组合元素释放强力连招！

### 元素系统

**元素相克:**
- 🔥 火克制💧水属性敌人（冰霜生物）
- 💧 水克制🔥火属性敌人
- ⚡ 雷克制💧水（电击效果）
- ❄️ 冰冻结和减速敌人
- 🌪️ 风推动和改变敌人方向

**连招魔法:**
- 🌋 蒸汽爆炸 = 火 + 水
- ⚡ 雷暴 = 雷 + 水
- 🌪️ 火焰龙卷 = 火 + 风
- 💎 水晶爆破 = 冰 + 雷
- 🌈 等离子炮 = 火 + 雷
- ...更多组合等你发现！

### 敌人类型

1. **🔥 火焰史莱姆** - 速度快，怕水
2. **💧 水元素** - 中等速度，怕雷
3. **⚡ 雷电灵** - 非常快，怕冰
4. **❄️ 冰霜巨人** - 速度慢，高血量，怕火
5. **🌪️ 风暴骑士** - 速度快，会偏转弱小魔法

### Boss战

每10波会遭遇一个强大的元素Boss，拥有独特的攻击模式和机制！
